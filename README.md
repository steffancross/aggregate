# Aggregate

A music listening app that brings together tracks from multiple platforms. Currently supports YouTube, SoundCloud, and Spotify.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) 15 with React 19
- **Database**: [Prisma](https://prisma.io) ORM with PostgreSQL
- **API**: [tRPC](https://trpc.io) for end-to-end type-safe APIs
- **Authentication**: [Clerk](https://clerk.com) for user authentication
- **Styling**: [Tailwind CSS](https://tailwindcss.com) 4.0
- **UI Components**: [Radix UI](https://www.radix-ui.com) primitives
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs)
- **Forms**: [React Hook Form](https://react-hook-form.com) with [Zod](https://zod.dev) validation
- **Testing**: [Vitest](https://vitest.dev)
- **Type Safety**: TypeScript
- **Built from the t3 stack**
